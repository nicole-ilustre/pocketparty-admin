## Getting Started

First, run the development server:

You would need `service-account.json` in folder root for application run

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

## Local Setup

- Please place service-account.json at the root of the application before running locally

## Deployment

- Please do not deploy this code as we do not have any securty mechanism

## Uploading Game Images

- Game images need to be added to nothing-pwa repo at it `./public/game-images` folder and committed and deployed
